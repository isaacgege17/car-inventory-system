# car-inventory-system
This is a beginner Python project that helps manage a car inventory through the terminal.
## Features 
- Add a car
- View all cars
- Delete a car
- User menu system
- Input validation
- Error handling using try/except
## Technologies Used
- Python
## What I Learned
This project helped me practice:
- loops
- functions
- lists
- dictionaries
- user input handling
- error handling
## How It Works 
The program displays a menu where users can:
1. Add a car to the inventory
2. View available cars
3. Delete a car
4. Exit the program
The inventory updates while the program is running.
